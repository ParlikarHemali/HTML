• Website:A collection of web pages identified by common domain name. There are 2 types of website dynamic (example facebook,twitter) and static (example resume,portfolio websites) All the websites together form the world wide web.
  • WWW:It stands for World Wide Web, it also stores websites and delivers it to users through the internet.
  • HTTP/HTTPs:Hyper Text Transfer Protocol is used to exchange or transfer data on www and Hyper Text Transfer Protocol Secure is a secure version of http it is also used to transfer data.
  • Client Server:A client is a user which requests data and server is a provider who provides the requested data to user.
  • Domain:It is a identity of a website or address with help of domain user can easily excess the website.
  • Host:A host is any computer connected to a network.
  • Hosting:It is an online service that makes your website's content accessible on internet.
  • FTP:It means File Transfer Protocol . FTP is used to transfer files between computers on a network.
  • path:It defines the location of a file or folder in computer's file system.
  • Extension:It is used to describe type of file name which is inserted at the end of file name. It helps the user to find the files quickly without opening it.
  • Editor / vscode:Editors are basically a computer program which is used to edit files. Vscode is a code editor.
  • URL-URI/URN:URL stands for Uniform Resource Locator. A URL is the address of a given unique resource on the Web(eg address).URI stands for Uniform Resource Identifier .It is a sequence of characters that distinguishes one resource from another (eg name). URN stands for Uniform Resource Name
  • Address bar:In a browser with the help of address bar we can search any website by typing name it also gives you the address or URL of the website.
  • Title bar:The title of a file or an application which is displayed on the top of window is called title bar.
