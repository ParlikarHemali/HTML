Text Formatting Tags

Start Tag End Tag Description
< b > < /b > It is used for bold text.
< strong > < /strong > It is used to define text with strong importance.
< i > < /i > It is used to make text italic in HTML.
< em > < /em > It is used to define emphasized text.
< mark > < /mark > It is used to mark or highlight text.
< del > < /del > It is used to delete the document.
< ins > < /ins > It is used to insert the document.
< small > < /small > It is used for smaller fonts.
< big > < /big > It is used for bigger fonts.
< sup > < /sup > It is used to add character above the normal line, and is sometimes rendered in a smaller font.
< sub > < /sub > It is used to rendered with a lowered baseline using smaller text.
